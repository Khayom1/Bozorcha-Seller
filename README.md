# Bozorcha Seller

Барнома барои фурӯшандагони маркетплейси Bozorcha (Тоҷикистон).

## Хусусиятҳо
- Регистратсияи фурӯшандагон (4 қадам)
- Тасдиқи шахсият тавассути акси паспорт
- Идоракунии маҳсулот
- Планҳои гуногун (FREE, BASIK, PRO, ULTRA)
- Реклама барои маҳсулот

## Саҳифаҳо
- Қадами 1: Рақами телефон
- Қадами 2: Рамзи SMS
- Қадами 3: Маълумоти шахсӣ
- Қадами 4: Акси паспорт ва доставка

## Технологияҳо
- Flutter
- Firebase (баъдтар)
